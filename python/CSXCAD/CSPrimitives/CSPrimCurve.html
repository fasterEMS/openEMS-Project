CSPrimCurve

class CSXCAD.CSPrimitives.CSPrimCurve

Bases: CSXCAD.CSPrimitives.CSPrimitives

Curve Primitive

A curve is a set of consecutive 3D coordinates.

Parameters
points(3, N) array

Array of 3D coordinates.

Examples

>>> x = np.array([0, 0, 1, 1]) + 1.5
>>> y = np.array([0, 1, 1, 0]) + 2.5
>>> z = np.array([0, 1, 3, 4])
>>> curve = CSPrimitives.CSPrimCurve(pset, metal, points=[x,y,z])
>>> curve.AddPoint([2, 0, 5])
