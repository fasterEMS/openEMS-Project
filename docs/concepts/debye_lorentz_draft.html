Definitions of other parameters are identical to the Drude model.

defines electrical susceptibility of a dielectric material as the following equation.

\[\chi(\omega) = \frac{\Delta \epsilon_{r}}{1+ j \omega t_\mathrm{relax}}\]

Its permittivity is given by:

\[\epsilon(\omega) = \epsilon_r(\infty) + \chi(\omega)\]

The first-order Debye model is given by:

\[\epsilon_r(\omega) = \epsilon_r(\infty) + \frac{\Delta \epsilon_{r}}{1+ j \omega t_\mathrm{relax}}\]

If the material has a non-zero DC conductivity \(\kappa\), an extra loss term appears.

\[\epsilon_r(\omega) = \epsilon_r(\infty) + \frac{\Delta \epsilon_{r}}{1+ j \omega t_\mathrm{relax}} - j \frac{\kappa}{\omega \epsilon_0}\]

By generalizing the Debye model to multiple oscillation poles, a high-order model is obtained. This is the model implemented in openEMS.

\[\epsilon_r(\omega) = \epsilon_r(\infty) + \sum^N_{p=1}{\frac{\Delta \epsilon_{r,p}}{1+ j \omega t_\mathrm{relax}(p)}} - j \frac{\kappa}{\omega \epsilon_0}\]

In terms of absolute permittivity:

\[\begin{split}\epsilon(\omega) = \epsilon_0 \left[ \epsilon_r(\infty) + \sum^N_{p=1}{\frac{\Delta \epsilon_{r,p}}{1+ j \omega t_\mathrm{relax}(p)}} - j \frac{\kappa}{\omega} \right] \\\end{split}\]

  • \(f\) is the frequency.

  • \(\epsilon_r(f)\) is the material’s relative permittivity.

  • \(\epsilon_r(\infty)\) is the material’s relative permittivity
